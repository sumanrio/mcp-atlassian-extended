## 📋 About mcp-atlassian-extended

mcp-atlassian-extended is a server tool made to work with Jira and Confluence. It offers 23 tools, 15 resources, and 5 prompts. The software helps teams organize sprints, manage agile boards, handle attachments, track calendars, plan time off, and calculate sprint capacity. It makes managing projects easier without needing technical skills.

This tool runs on Windows and connects with common Atlassian products. Whether you work in a small team or a large project, the tools help you keep track of your work and deadlines.

## 🔍 Key Features

- Sprint planning and tracking  
- Agile board management  
- Attachments handling within tasks  
- Team calendar and time-off tracking  
- Sprint capacity calculations  
- Integration with Jira and Confluence  
- Easy-to-use interface for project managers  
- Supports AI-driven task management
